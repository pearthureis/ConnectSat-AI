# ConnectSat AI

## Sobre o projeto

O ConnectSat AI é um sistema desenvolvido em Python que simula o monitoramento de um satélite responsável por fornecer internet para áreas rurais.

O sistema gera dados de telemetria, identifica possíveis falhas automaticamente e utiliza Inteligência Artificial para analisar a situação e explicar os impactos para a população.

## Funcionalidades

- Simulação de telemetria do satélite
- Detecção automática de alertas
- Análise inteligente da missão
- Exibição organizada das informações no terminal

## Parâmetros monitorados

- Latência
- Throughput
- Temperatura do transponder
- Qualidade do sinal

## Tecnologias utilizadas

- Python 3
- Ollama
- IA Generativa
- Git e GitHub

##  Como executar

1. Clone o repositório.
2. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```
3. Execute:
   ```bash
   python main.py
   ```
